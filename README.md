# X-ray Sensor Parameter Calculator

A static single-page dashboard for X-ray NDT sensor parameter estimation.

The dashboard calculates:

- SID
- Tube current
- Object transmittance
- Dose rate
- Scintillator light intensity
- Sensor photocurrent
- Dose-to-current efficiency
- Integration time range

Calculated results are based on Excel-derived experimental calibration models.

Maintained for DADS lab internal research and demonstration use.
